Ascii-Logo-Studio is a professional Java application that allows you to convert text into various ASCII Art styles. This project is developed using **Java 25** and designed to be simple and user-friendly.

## Features

- Supports multiple ASCII styles:
  - Block Style (2D and 3D)
  - Star Style
  - Outline Style
  - Thin ASCII
  - Matrix Pattern
  - Bubble Style
  - Letter Fill (Uppercase and Lowercase)
  - Display all styles at once
- Choose between **uppercase or lowercase** letters in Letter Fill style.
- Choose **2D or 3D** in Block style.
- Quick execution using **`java-run.bat`** file.
- Interactive console for entering text and selecting styles.
- Option to show **default examples** of all styles.

## Requirements

- **Java 25** must be installed on your system.
- Windows OS (for using the .bat file).

## How to Run

1. Download and install **Java 25**.
2. Run the **`java-run.bat`** file in the project.
   - This file automatically compiles and runs the program.
3. After running, select the desired style and enter your text.
4. The ASCII result will be displayed in the console.
